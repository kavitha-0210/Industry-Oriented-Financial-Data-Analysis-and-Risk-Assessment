# Industry-Oriented-Financial-Data-Analysis-and-Risk-Assessment
This project analyzed financial customer data to identify risk patterns and support financial decision-making. Key financial indicators such as income, loan amount, credit score, and debt-to-income ratio were examined.

Customers with lower credit scores and higher debt-to-income ratios were identified as higher-risk individuals. A Logistic Regression model was used to classify customer risk levels and assess future risk potential.

The project demonstrates how financial analytics can support credit evaluation, risk management, and informed lending decisions while reducing financial losses.
